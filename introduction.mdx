# Welcome to Bodequita

**Bodequita** is a mobile application built for Android that helps local store owners manage their day-to-day operations — from tracking inventory and recording sales to monitoring purchases and closing the cash register at the end of the day.

Built with **Kotlin**, **Jetpack Compose**, and **Room**, Bodequita delivers a clean Material Design 3 experience with a robust MVVM architecture that keeps your data safe on-device.

## Key highlights

- **100% offline** — All data is stored locally using Room (SQLite). No internet connection required.
- **MVVM architecture** — Clean separation of concerns using ViewModels, Repositories, and DAOs.
- **Material Design 3** — Modern, accessible UI components built with Jetpack Compose.
- **Real-time stock control** — Stock levels update automatically when you record sales or purchases.
- **Dark/Light theme** — Switch between themes from the profile screen.
- **Charts & analytics** — Weekly activity charts powered by the Vico library.
